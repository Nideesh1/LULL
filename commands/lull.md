---
description: Turn Lull ads on or off in your Claude Code status line
---

Wire (or unwire) Lull's status-line ads by running its installer.

- If the user's argument is `off`, run: `lull init --off`
- Otherwise, run: `lull init`

After it succeeds, tell the user to open a **new** Claude Code session to see
the change, and that ads are ⌘-click (Ctrl-click on Linux/Windows) to open.
